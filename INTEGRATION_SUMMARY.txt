Transaction Monitoring Workflow:
┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐  ┌────────┐
│ Upload │→│ Parse  │→│ Enrich │→│  Check │→│  Alert │→│ Review │

7 Built-in AML Rules:
✓ Prohibited Country      - CRITICAL severity
✓ High-Risk Country       - HIGH severity
✓ Large Cash Deposit      - HIGH severity
✓ Large Cash Withdrawal   - HIGH severity
✓ Outlier Detection       - MEDIUM severity
✓ Velocity Alert          - MEDIUM severity
✓ Unusual Narrative       - LOW-MEDIUM severity

Alert Severity Levels:
CRITICAL (100)  → Prohibited country, blocks transaction
HIGH (75-90)    → Significant risk, requires immediate review
MEDIUM (50-70)  → Elevated risk, review within 48 hours
LOW (25-45)     → Minor concern, routine review
INFO (10-20)    → Informational only, no action required
