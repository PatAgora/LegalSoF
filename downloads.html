🚀 Quick Start Testing

  • 1. Download Client Info Template and Simple Bank Statement
  • 2. Open the SoF Assessment System
  • 3. Navigate to: Matters → REF-2024-001 → SoF Assessment tab
  • 4. Upload Client Info (or enter manually)
  • 5. Upload Bank Statement CSV
  • 6. Click 🚀 Run SoF Assessment
  • 7. Review results: claims extraction, evidence matching, funding paths, red flags, and Transaction Review integration
  • 8. Download the audit file note

💡 CSV Format Quick Reference

  • Required Columns: account_id, date, amount, currency, direction, description
  • Optional Columns: counterparty_name, balance
  • Date Format: YYYY-MM-DD (e.g., 2023-06-15)
  • Amount Format: Positive numbers only, no currency symbols (e.g., 250000.00)
  • Direction: "credit" for money IN, "debit" for money OUT
  • Currency: Use GBP for UK transactions
